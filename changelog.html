What's New

Every update to Receipts for YNAB, from launch to today. Built by a solo developer and fellow YNAB budgeter, shaped by user feedback.

v1.0.9 April 21, 2026 Latest

Better scanning for long receipts and stability improvements.

  • New Long receipt scanning — Grocery hauls and other tall receipts now scan more accurately with adaptive processing that breaks them into optimized sections
  • Improved YNAB merchant names — Long merchant names are now handled gracefully when syncing to YNAB
  • Fixed Upgrade stability — Resolved a crash that could occur when updating from older versions of the app
  • Improved Scan cancellation — Cancelling a scan mid-process is now faster and more reliable
v1.0.8 March 17, 2026

Fractional quantities, gas station receipts, and scanning accuracy improvements.

  • New Fractional quantities — Enter weighted items like 0.335 kg of produce. Quantities are no longer limited to whole numbers
  • New Gas station receipts — Dedicated detection for fuel receipts. Correctly extracts fuel grade, total, and ignores volume/unit price
  • Improved Price extraction accuracy — Fixed currency values being split across lines (e.g., "$85." and "50" now correctly read as $85.50)
  • Improved Quantity accuracy — The AI no longer fabricates quantities by merging same-named items. Each receipt line is extracted individually
  • Fixed Upload to YNAB button — Redesigned toolbar for easier access. Now always visible for local receipts, with a helpful prompt if no account is selected
  • Improved Accessibility — Better VoiceOver grouping for line items, improved notification announcements, and Reduce Motion support throughout
v1.0.7 March 3, 2026

AI model status, higher accuracy, and a simplified save flow.

  • New AI Model Status Indicator — A card tells you when Apple Intelligence is still downloading, with a success toast when the model is ready
  • Improved Scanning accuracy — Neural Engine document detection with perspective correction. Overall accuracy up from 73% to 85%, price extraction from 86% to 98%
  • Improved Save flow simplified — Receipts now always save locally first. Upload to YNAB when you're ready from the receipt detail view
  • Fixed Bilingual receipts — IKEA and other multilingual receipts no longer blocked by language detection
  • Improved Accessibility — WCAG AA contrast fixes, VoiceOver improvements for notifications and action cards, Reduce Motion support
v1.0.6 February 18, 2026

Complete Settings redesign and Liquid Glass polish.

  • New Settings redesign — Reorganized into clean sub-pages: YNAB Account, Scanning & AI, Subscription, Data & Storage, and About
  • Improved Liquid Glass polish — Tab bar and toolbar transitions now use proper glass-aware styling
  • Fixed Subscription purchases — Fixed a bug where purchase buttons could stop responding. Resubscription for returning users now works correctly
  • Fixed Localization — Fixed 408 entries that could display raw keys instead of English text
  • Improved Accessibility — 30+ VoiceOver, Switch Control, and Dynamic Type improvements across the entire app
v1.0.5 February 11, 2026

Multi-budget awareness and a better receipt browsing experience.

  • New Multi-budget awareness — Budget name shows under the Receipts title. Filter receipts by budget. Budget and account names saved with each receipt
  • New Connection status badge — Orange indicator on Settings tab when using cached data offline
  • Improved Accounts & Categories — Redesigned selection views with better layout
  • Improved Amount fields — Text selects all on tap with visible currency symbol
  • Fixed Line item amount doubling — Syncing from YNAB no longer doubles amounts when quantity is greater than 1
v1.0.4 February 4, 2026

Bank import matching and smarter error handling.

  • New Bank import matching — Receipts automatically match with bank imports in YNAB. No more duplicate transactions cluttering your budget
  • New Smart error messages — YNAB API errors now show clear explanations with one-tap actions: edit the receipt, reconnect YNAB, or retry automatically
  • New Auto-retry for rate limits — If YNAB is busy, the app retries automatically with backoff
  • Fixed Paywall purchase buttons — Fixed a regression where subscription purchase buttons stopped responding
v1.0.3 January 27, 2026

On-device category learning and smarter tax handling.

  • New Category learning experience — Progress badge shows how the on-device learning model is training on your preferences. First-scan tooltip explains how it works
  • New Category groups — Pickers now mirror your YNAB category group ordering instead of flat alphabetical lists
  • Improved Tax handling — Clearer explanation when item totals don't match the receipt total (tax, fees, and rounding differences)
  • Fixed New line item defaults — New line items now start at $0.00 instead of the receipt total
  • Fixed Sheet dismiss protection — Receipt verification can no longer be accidentally dismissed by tapping outside
v1.0.2 January 19, 2026

Improved onboarding experience.

  • Improved Onboarding redesign — New messaging that better explains receipt splitting, privacy-first processing, and the value of AI scanning
v1.0.1 January 15, 2026

On-device category learning and offline access.

  • New On-device category learning — The app learns your categorization preferences using on-device AI. Predictions get smarter with every receipt you categorize
  • New Offline category access — Categories and accounts remain available even when your YNAB session expires. Scan and categorize offline, sync later
  • New Glass tile animations — Beautiful glass-lucent scan action tiles with gyroscopic tilt and sparkle effects
  • New Smart scan limits — Prorated free tier when a subscription expires mid-month, so you always get fair usage
v1.0.0 January 2026

Initial release.

  • New AI-powered receipt scanning — Scan receipts with Apple Intelligence. Extracts merchant, date, total, and individual line items automatically
  • New YNAB sync — One-tap sync to create split transactions in YNAB with line items as subtransactions
  • New 100% on-device processing — All scanning happens on your device using Apple Intelligence. No cloud uploads, no data collection
  • New Receipt vault — Store and browse receipts locally for warranties, returns, and expense tracking
  • New Confidence scoring — See how accurate each scan is before syncing, with the option to review and edit
  • New Multiple capture modes — Camera, photo library, or document scanner
